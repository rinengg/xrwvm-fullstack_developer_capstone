# Best Cars Dealership - Full Stack Capstone Project

## Repository Name
xrwvm-fullstack_developer_capstone

## Project Name
Best Cars Dealership Application

## Description
A full-stack web application for a car dealership platform built with Django, React, Node.js/Express, and MongoDB. The application allows users to browse dealerships, read and post reviews, and manage car inventory through a Django admin interface.

## Tech Stack
- **Frontend**: React.js with React Router
- **Backend**: Django (Python)
- **Database**: SQLite (Django), MongoDB (Express)
- **Microservices**: Node.js/Express (dealer data), Flask (sentiment analysis)
- **CI/CD**: GitHub Actions

## Features
- User registration and authentication
- Browse car dealerships by state
- View dealer details and customer reviews
- Post reviews with sentiment analysis
- Admin panel for managing car makes and models
